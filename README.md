# Admin Stock

Sistema de gestión de inventario desarrollado con Laravel + React + TypeScript.

## Características

- Gestión de productos
- Gestión de stock
- Ventas
- Roles y permisos
- Dashboard
- Reportes

## Tecnologías

- Laravel 12
- React
- TypeScript
- Inertia
- Tailwind CSS
- MySQL

## Instalación

composer install
npm install
php artisan migrate
npm run dev
